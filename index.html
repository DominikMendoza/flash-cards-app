## Explain how a for loop works in JavaScript
A for loop is used to repeat a section of code a specific number of times.
%
A for loop in JavaScript has the following structure:

```javascript
for (initialization; condition; increment) {
  // code to execute
}
```

Where:
- Initialization: executed before the loop starts
- Condition: defines when the loop should continue
- Increment: executed after each iteration

Getting Started

  1. Upload a Markdown file or paste Markdown content
  2. Click "Generate Flashcards" to create your cards
  3. Use the navigation buttons to move between cards
  4. Click "Flip Card" to see the answer
  5. Mark your answer as correct or incorrect

About Markdown Flashcards

Markdown Flashcards is a simple web application that allows you to create and study flashcards from Markdown files.

This application was created as an educational project to demonstrate how to build a flashcard system using HTML, CSS, and JavaScript.
