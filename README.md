# Flashcard Markdown Format

1. **Basic Card**:

   ```
   ## Question
   Answer
   ```

2. **True/False Card**:

   ```
   ## [T/F] Statement
   F
   Explanation
   ```

3. **Multiple Choice Card**:

   ```
   ## Question
   - [ ] Incorrect option
   - [x] Correct option
   - [ ] Incorrect option
   - [ ] Incorrect option

   Explanation
   ```

4. **Concept/Definition Card**:

   ```
   ## Concept: Term
   Definition
   ```

5. **Card with extended answer**:
   ```
   ## Question
   Short answer
   %
   Detailed answer with multiple lines
   ```
